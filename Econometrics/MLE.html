MLE

likelihood function

給定參數 \(\theta\)，令隨機變數 \(y\) 的 PDF 函數為 \(f(y|\theta)\)。\(f(y|\theta)\) 可以用來描述資料是如何產生的。若有 \(n\) 個 i.i.d. 的樣本來自同一個 PDF 函數，則其聯合機率密度函數為 \[f(y_1...y_n|\theta) = \prod_{i=1}^n f(y_i|\theta)\]

給定 \(\theta\)，此機率密度函數用來描述出現 \(\{y_i \}\) 樣本的機率。我們也可以反過來詮釋，給定我們觀察到的樣本為 \(\{y_i \}\)，則給予不同的 \(\theta\) 值，可以求得不同的 \(f(y_1...y_n|\theta)\) 值，因此我們可以問，哪一個 \(\theta\) 值，能讓 \(f(\cdot|\theta)\) 值最大，意即讓樣本出現 \(\{ y_i\}\) 的機率最大，此即最大概似的概念。根據此概念，我們可以定義概似函數 (likelihood function) 為 \[L(\theta|y_1...y_n) = \prod_{i=1}^n f(y_i|\theta) = f(y_1,..., y_n|\theta)\]

\(L(\theta|y_1...y_n)\) 跟 \(f(y_1...y_n|\theta)\) 其實是一模一樣的函數形式，差別只是在詮釋上的不同。對此概似函數取 \(\ln\)，即得到 log-likelihood function: \[\ln L(\theta|y_1...y_n) = \sum_{i=1}^n \ln f(y_i|\theta)\]

我們可以將 log-likelihood function 一般化，寫成 \(\ln L(\theta| Data)\)，並求解 \[\max_{\theta} \ln L(\theta|Data)\] 則其解 \(\widehat\theta\) 即為最大概似估計式(MLE)。

MLE 的性質

根據 Greene 計量課本第 5 版第 17 章，若 \(f(y_i|\theta)\) 滿足定義 17.3，則 MLE 具有定理 17.1 的性質。定義 17.3 大致上要求 \(\ln f(y_i|\theta)\) 的一階導數、二階導數、三階導數必須有期望值，且為連續函數。令 \(\theta_o\) 表示真正的參數值，\(E_o[.]\) 表示給定真正參數值時的期望值，則定理 17.1 包括以下幾點

  1. MLE 具有一致性(consistency): 對任意小的正實數 \(\epsilon\)，\[\lim_{n\rightarrow\infty}P(|\widehat\theta_n - \theta_o|\leq \epsilon) = 1,\]
  2. MLE 具有漸進常態性(asymptotic normality): \(\widehat\theta \sim N[\theta_o, I(\theta_o)^{-1}]\)，其中 \[I(\theta_o) = -E_o\left[\frac{\partial^2 \ln L(\theta_o) }{\partial\theta_o\partial\theta_o'}\right]\] 稱為 information matrix，為 log-likelihood function 的 Hessian matrix 取期望值加負號。

如何估計 covariance matrix

方法一: 若 log-likelihood function 的二階導數的函數形式已知，則可以直接將 \(\widehat\theta\) 代入公式，取期望值。意即直接算出 \[I_1(\theta_o) = -E_o\left[\frac{\partial^2 \ln L(\widehat\theta) }{\partial\widehat\theta\partial\widehat\theta_o'}\right]\]

方法二: 另一種估計方法是將 \(\widehat\theta\) 代入 log-likelihood function，算出實際的二階導數，意即計算 \[I_2(\widehat{\theta}) = -\left( \frac{\partial^2 \ln L(\widehat\theta) }{\partial\widehat\theta\partial\widehat\theta_o'} \right)\]

方法三: 第三種估計方法是計算一階導數的 covariance matrix，意即計算 \[I_3(\widehat{\theta}) = \sum_{i=1}^n \widehat{\bf{g}}_i \widehat{\bf{g}}'_i\] \[\widehat{\bf{g}}_i = \frac{\partial\ln f(x_i, \widehat\theta)}{\partial\widehat\theta}\]
