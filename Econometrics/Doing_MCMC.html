以下參考 Computational Statistics in Python 說明如何用 Python 的 scipy 套件實作 MCMC 的貝氏推論。我們以擲銅板的實驗為例, 說明如何設定先驗分配, 設定 likelihood, 計算後驗分配, 以及如何使用 Metropolis-Hastings 演算法。我們採用的先驗分配是 Beta 分配, 且擲銅板實驗的 likelihood 為二項分配, 所以可以得到後驗分配的解析解, 且該解析解也是 Beta 分配。

解析解¶

  • 先驗分配: $\pi(\theta)\sim Beta(a, b)$
  • 擲銅板 $n$ 次, 其中有 $h$ 次出現正面, 則 likelihood: $f(h|\theta, n)\sim Bin(n, \theta)=\tbinom{n}{h}\theta^h(1-\theta)^{n-h}$
  • 後驗分配: $\Pi(\theta|n, h)\sim Beta(h+a, n-h+b)$

import matplotlib.pyplot as plt

    return st.binom(n, thetas)

Metropolis-Hastings 演算法¶

  • 本期狀態: $\theta_t$
  • 候選變量: 抽出 $\theta_p\sim N(\theta_t, \sigma)$
  • 計算 $\rho$ $$\rho=\frac{\Pi(s_p|D)}{\Pi(s_t|D)}=\frac{f(D|s_p)\pi(s_p)}{f(D|s_t)\pi(s_t)}$$
  • 建立 $\theta_{t+1}$
    • 若 $\rho \geq 1$, 則 $\theta_{t+1} = \theta_p$
    • 若 $\rho < 1$, 則抽出 $u\sim U(0, 1)$, 若 $u\leq \rho$, 則 $\theta_{t+1} = \theta_p$, 反之則 $\theta_{t+1}=\theta_t$
  • 當 $t$ 夠大, 統計 $\theta$ 的分布

# 取得資料
n, h = 100, 61 

# 設定先驗分配
a, b = 10, 10
prior = st.beta(a, b) 

# 設定 likelihood 
def lik(n, thetas): 
    return st.binom(n, thetas)

# 設定 likehood * 先驗分配 
def target(lik, prior, n, h, theta):
    if theta < 0 or theta > 1:
        return 0
    else:
        return lik(n, theta).pmf(h)*prior.pdf(theta)
    
# 設定 theta_p 所需要的常態分配參數
sigma = 0.1

# 設定抽樣次數
niters = 30000

# 設定 Markov chain
samples = np.zeros(niters+1)

# 設定目前 theta
theta = 0.5
samples[0] = theta

# 建立 Markov chain
for i in range(niters):
    theta_p = st.norm(theta, sigma).rvs()
    rho = min(1, target(lik, prior, n, h, theta_p)/target(lik, prior, n, h, theta))
    u = np.random.uniform()
    if u <= rho:
        #naccept = naccept+1
        theta = theta_p
    samples[i+1] = theta
    
# 只選取第 nmcmc 期後的資料計算 \theta 的分配
nmcmc = len(samples)//2

# 建立 Markov chain
def gen_markov_chain(niters, n, h, theta, lik, prior, sigma):
    samples = np.zeros(niters+1)
    samples[0] = theta
    for i in range(niters):
        theta_p = st.norm(theta, sigma).rvs()
        rho = min(1, target(lik, prior, n, h, theta_p)/target(lik, prior, n, h, theta))
        u = np.random.uniform()
        if u <= rho:
            #naccept = naccept+1
            theta = theta_p
        samples[i+1] = theta
    return samples
