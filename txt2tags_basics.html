txt2tags Basics

1. 編輯 .t2t 文件

  • 使用任何文字編輯軟體編輯文件, 存檔時, 附檔名必須為 .t2t。
  • 假設我們將檔案存在 MyFolder 資料夾，檔名為 FileName.t2t。

2. 將 .t2t 輸出成 .html

  • 開啟 Terminal。
  • 如果不在 FileName.t2t 所在的資料夾，則切換過去。
  • 執行 txt2tags -t html FileName.t2t
  • 若正常執行, 則 MyFolder 資料夾會出現 FileName.html 檔案。

3. 將 .html 放在網路上

  • 先申請 Github pages。可參考此網頁說明。
  • 假設我們申請的 Github 帳號為 GitAccount, Github repository 名稱為 GitRepo。
  • 將 FileName.html 放到本機端的 GitRepo 資料夾。
  • 開啟 GitHub Desktop。
  • 輸入 commit 後, 執行 Commit to gh-pages。
  • 執行 sync。
  • 用瀏覽器開啟 https://GitAccount.github.io/GitRepo/FileName.html, 應該可以看到我們編輯好的網頁。
