oTree

install

$pip3 install --upgrade otree-core
如果出現 'permission denied', 輸入
$sudo pip3 install --upgrade otree-core

start project

$otree startproject your_project_name

start app

先進入 project 資料夾, 輸入
$otree startapp your_project_name

資料庫與伺服器

$otree reserdb
$otree runserver

使用 Heroku

$git add .
$git commit -am 'my comment'
$git push heroku master
$heroku run otree resetsb ## 重設資料庫
$heroku open
