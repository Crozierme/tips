權重初始值設定

  • 原則上，隱藏層的活性化分布要越多元越好，且要避免梯度消失(微分值為0)的狀況。如果某隱藏層有 100 個神經元，但活性化分布幾乎集中在同一個值，就表示一個神經元就足以表示這 100 個神經元。
  • 如果 activation function 為 sigmoid 函數，則權重不能太大，否則活化的結果不是 0 就是 1。但是也不能太小，否則活化的結果幾乎都是 0.5。 Xaiver 建議如果上層節點有 nnn 個，則以標準差為 1/n\sqrt{1/n}1/n​ 的常態分配分布初始化權重。
  • 如果 activation function 為 ReLU 函數，則活化結果只保留正值，負值都會被 0 取代，因此活性化分布會漸漸集中於 0，因此權重值要更多樣化。何恺明建議如果上層節點有 nnn 個，使用標準差為 2/n\sqrt{2/n}2/n​ 的常態分配分布初始化權重。

CNN

  • convolution: 以 2D 圖形辨識為例，假設 input 為一個 25 x 25 的 2D 圖形，在 affine 層，我們會設定 input 層有 625 個神經元，這種作法等於是把 2D 的圖形當成一維的向量處理。在 convolution 層，我們還是把 2D 圖形當 2D 圖形看待，透過另一個較小的 2D 濾鏡(又稱為 kernel)，上下左右移動，篩出圖形的局部特徵(feature)。
  • padding: 資料的周圍進行補 0。
  • stride: 濾鏡每次移動的步幅。
  • pooling: 減少資料維度。
