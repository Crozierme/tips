1. 開始之前

  • 使用任何文字編輯軟體編輯文件, 存檔時, 副檔名必須為 .t2t。
  • 一份 t2t 文件可以分成三大區塊： Header, Config, Body。Header 固定使用文件的前三行，緊接著是 Config，然後是 Body。

2. Header

  • 第 1 行固定為文件標題
  • 第 2 行為斜體字, 可放作者或 email。
  • 第 3 行為一般字體, 可放修改日期或版本資訊。
  • 如果要自動生成文件的修改日期，請加上 %%date(%Y.%m.%d)

3. Configration

  • 可以指定 t2t 如何轉換成其他格式。指令固定為 %!keyword: value
  • %!target: html: 指定將文件轉換成 .html
  • %!options: --toc --toc-level 3: 指定轉換時加上目錄，並指定目錄層級。
  • %!encoding: UTF-8: 指定轉換時的編碼方式。
  • %!style: fancy.css: 指定轉換後的風格。
  • 更多說明請見 Settings, Configuration
