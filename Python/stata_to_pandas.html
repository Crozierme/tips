  • df.drop_duplicates(keep = ...)

    • keep = "first": 保留第一次出現的紀錄
    • keep = "last": 保留最後一次出現的紀錄
    • keep = False: 刪除所有重複出現的紀錄

  • df.duplicated(keep = ...)

    • keep = "first": 除了第一次出現的紀錄外，都標示為複本
    • keep = "last": 除了最後一次出現的紀錄外，都標示為複本
    • keep = False: 所有記錄都標示為複本

merge

  • stata: merge m:1 keys using usingdata

  • pandas: pd.merge(leftdf, rightdf, how = 'outer', on = [keys])

  1. stata 的 merge 必須註明 m:1 或 1:m， pandas 不用。
  2. how = 'left', 只保留來自 leftdf 的資料
  3. how = 'right', 只保留來自 rightdf 的資料
  4. how = 'inner', 保留同時出現在 leftdf 與 rightdf 的資料
  5. how = 'outer', 保留所有資料
