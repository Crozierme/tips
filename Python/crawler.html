抓取網頁

概念

  • 網路爬蟲分成兩個部分: 擷取網頁內容，分析網頁內容。
  • 網頁又可以分為靜態網頁與動態網頁。靜態網頁指純文字與圖片的網頁，沒有任何程式功能，不會讀取資料庫的資料。動態網頁是會與使用者互動的網頁，使用者與網頁互動後，網頁會透過程式與資料庫結合，決定所要呈現的內容。
  • 簡單來說，靜態網頁的內容是固定的，不同使用者在相同的網址看到的內容都一樣。但是動態網頁的內容是動態的，相同的網址呈現的內容不一定相同。
  • 動態網頁中有一類為 ASPX 網頁，它可以讓使用者在網頁上填資料，發送表單並查詢。這些表單資訊會存成 ViewState 以及 EventValidation 兩個變數，爬蟲時需要加上這兩個變數的值，才能模擬發送表單的動作。

擷取靜態網頁

  • 使用 requests 取得網頁原始碼
  • 基本流程：
import requests

url = 'https://www.ptt.cc/bbs/movie/index.html'
response = requests.get(url) 
source_text = response.text # 取得網頁原始碼
print(source_text)

擷取動態網頁

  • 使用 selenium 取得網頁原始碼
  • 如果我們用 requests 取得網頁原始碼，會發現原始碼並沒有動態網頁呈現的內容。這是因為動態網頁的內容是由程式運算後從資料庫取得，不像靜態網頁是固定的內容。
  • 基本流程：
from selenium import webdriver

url = 'http://24h.pchome.com.tw/region/DRAA'
driver = webdriver.Firefox()
driver.get(url)
source_text = driver.page_source # 取得網頁原始碼
print(source_text)
driver.quit()

發送 ASPX 表單

  • 先在瀏覽器上發送表單，並取得 cookie 內容，包括 ASP.NET_SessionId、__VIEWSTATE、__EVENTVALIDATION，再將這些參數值放入程式裡開始爬蟲。
  • 基本流程
import requests

url = "https://www1.nhi.gov.tw/mqinfo/SearchPro.aspx?Type=AMI&List=4"
params = {   
    "__EVENTVALIDATION":"/wEPDwUKMTA5ODI3",
    "__VIEWSTATE":""/wEdAJYB3w0oMFwGoAK",
    }
mycookies = {"ASP.NET_SessionId": ydbyi00mgkixuuia1wu5rzmr}

response = requests.post(url, data = params, cookies = mycookies) #發送表單
source_text = response.text # 取得網頁原始碼
print(source_text)

分析網頁 (parsing)

  • 可使用 lxml
  • 基本流程:
from lxml import etree

html = etree.HTML(source_text) # 將之前擷取的網頁原始碼轉成 etree 元素

抓取圖片

  • 基本流程
import requests
from PIL import Image
from io import BytesIO

img_url = 'https://i2.bahamut.com.tw/baha_logo5.png'
response = requests.get(img_url)
img = Image.open(BytesIO(response.content))
img.save('img.png')
