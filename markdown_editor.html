ATOM

如何安裝 ATOM 套件？

  1. 上方選單列點選 ATOM -> 偏好設定
  2. 點選新增套件或主題
  3. 搜尋套件, 點選安裝

如何檢視與設定已安裝 ATOM 套件？

  1. 上方選單列點選 ATOM -> 偏好設定
  2. 點選已安裝的擴充套件

Markdown Preview Enhanced 套件

  • 此套件功能相當強大, 支援 latex 語法, code chunk, 繪製流程圖, 自動生成目錄, 預覽及匯出 html 檔案, 請參考官方文件。以下介紹幾個常用功能:
如何自動生成目錄？
  1. 安裝 Markdown Preview Enhanced 套件
  2. 按下 cmd + shift + p
  3. 搜尋 toc
  4. 點選 Markdown Preview Enhanced: Create Toc
  5. 存檔, 此時文件上方會出現 toc, 可以在裡面調整設定
  6. 不想被加入目錄的標題, 可以在標題後方加入 {ignore=true}
如何預覽及輸出 html 格式？
  1. 安裝 Markdown Preview Enhanced 套件
  2. 上方選單列點選擴充套件
  3. 在 Markdown Preview Enhanced 點選 Toggle preview
  4. 在預覽畫面按右鍵, 點選 HTML -> HTML(offline)
如何在存檔時自動輸出 html 格式？
  • 在文件開頭加入以下代碼
    ---
export_on_save:
  html: true
---
執行 code chunk

  • ATOM 的 code chunk 除了可以依照程式語言對程式碼上色外, 還可以執行程式碼, 將結果呈現在 html 上。譬如輸入以下程式碼:

      	```python {cmd=true matplotlib=true}
  	import matplotlib.pyplot as plt
  	plt.plot([1,2,3, 4])
  	plt.show() # show figure
  	```

    即可產生以下圖檔

    若要啟用執行程式功能, 進入 Markdown Preview Enhanced 設定區, 將 Enable Script Execution 開啟。特別注意, 因為此功能會自動執行文檔內的程式碼, 要提防惡意或錯誤的程式碼造成的資安風險。

obsidian

如何在 obsidian 文件建立動態目錄？

  1. 安裝 Dynamic Table of Contents 插件
  2. 在想要插入目錄的位置, 插入以下代碼, 即可產生目錄
	```toc
	   title: "## 目錄"
	```
  1. 更多設定請見官方文件

如何安裝 obsidian 插件？

  1. 點左下角齒輪
  2. Community plugins 取消 Safe mode
  3. 點選 Browse, 搜尋插件
  4. 點選 install 安裝插件
  5. 在 installed plugins 啟用及設定插件

mention(提及) 與連結

  • 在 A 筆記介面的最右側, 在 Unliked mentions 區塊會自動顯示有哪些筆記提及了 A 筆記的檔案名稱。
  • 如果要在 A 筆記提及 B 筆記, 有以下幾種方式：
    1. 輸入 [[B 筆記檔案名稱]], 建立與 B 筆記的連結。
    2. 輸入 [[B 筆記檔案名稱#]], 系統會提示 B 筆記出現的標題, 可連結到該標題。
    3. 輸入 [[B 筆記檔案名稱^]], 系統會提示 B 筆記的文章區塊, 可連結到該區塊。
  • 在最左側的 Open Graph View 可以看到筆記之間的連結關係。
